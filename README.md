# Customer Churn Analysis Using Power BI.

## Project Overview
This project analyzes customer churn behavior for a telecom company using Power BI.
The goal is to identify key factors contributing to customer churn and provide actionable insights to improve customer retention.

## Tools & Technologies
- Power BI
- DAX
- Microsoft Excel

## Dataset
- Telco Customer Churn Dataset
- Records: 7,043 customers

## Key Metrics
- Total Customers: 7,043
- Churn Rate: 26.54%
- Churned Customers: 1,869

## Key Insights
- Month-to-month contract customers show the highest churn rate.
- Customers with higher monthly charges are more likely to churn.
- New customers (0â€“12 months tenure) have the highest churn risk.
- Long-term customers (49+ months) show strong retention.

## Business Recommendations
- Promote long-term contracts through discounts.
- Improve onboarding for new customers.
- Optimize pricing strategies for high-paying customers.

## Dashboard Features
- KPI Cards
- Churn by Contract Type
- Churn by Tenure Group
- Monthly Charges vs Churn
- Interactive Slicers
